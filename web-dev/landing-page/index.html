Proyectos

Predicción de precios de Bitcoin con Machine Learning

Este proyecto aplica técnicas de ciencia de datos para predecir si el precio de Bitcoin subirá, utilizando datos obtenidos desde la API de CoinGecko. Se entrenaron modelos de clasificación como Decision Tree y Random Forest, con una precisión final del 93%.

  • Tecnologías: Python, Pandas, scikit-learn, Matplotlib, API REST
  • Modelo entrenado: Clasificación binaria con evaluación cruzada
  • Resultado: Precisión promedio del 93%
